# StarWarsApp

Android приложение для просмотра персонажей вселенной Star Wars на основе открытого API [swapi.dev](https://swapi.dev).

## Функциональность

- Список персонажей с основной информацией
- Поиск персонажей по имени
- Детальный экран с полной информацией о персонаже
- Кэширование данных — приложение работает без интернета
- Индикатор загрузки при запросе данных
- Сообщение если данных нет

## Архитектура

Проект построен по принципам **Clean Architecture** с разделением на три слоя:

```
presentation/   — UI (Jetpack Compose), ViewModel, UiState
domain/         — бизнес-логика: модели, репозитории (интерфейсы), UseCase
data/           — реализация: Retrofit, Room, маппинг данных
```

Навигация между экранами реализована через **Navigation Compose** с типобезопасными маршрутами через `sealed class`.

Кэширование реализовано в репозитории: при наличии интернета данные загружаются из сети и сохраняются в Room, при отсутствии — возвращаются из локальной базы.

## Технологии

| Технология | Назначение |
|---|---|
| Kotlin | язык разработки |
| Jetpack Compose | декларативный UI |
| Hilt | dependency injection |
| Retrofit + OkHttp | сетевые запросы |
| Room | локальное кэширование |
| Navigation Compose | навигация между экранами |
| ViewModel + StateFlow | управление состоянием UI |
| Coroutines | асинхронная работа |

## Запуск

1. Клонировать репозиторий
2. Открыть в Android Studio
3. Запустить на устройстве или эмуляторе (minSdk 24)
